# bpmn-viewer-vue
> 工作流流程渲染组件

![bpmn-viewer-vue](https://dev.dpark.com.cn/iplatform/codimd/uploads/upload_e649630e2412b862d9c9314def7635b2.gif)

[在线DEMO](http://192.168.126.25/flowable-editor/#/bpmn/staticViewer)

## 特点
* 优化默认流程图样式。
* 增加节点动效，描述不同的流程状态。
* 增加时间轴侧边栏，显示更多详细信息。
* 增加时效与持续时间属性。  
* 增加时间轴与流程图联动效果。
