# bpmn-viewer-vue
> 基于bpmn2.0标准的工作流流程图渲染组件

![bpmn-viewer-vue](http://img.xdo.icu/bpmn-viewer-vue.gif)

[在线DEMO](http://192.168.126.25/flowable-editor/#/bpmn/staticViewer)

## 特点
* 优化默认流程图样式。
* 增加节点动效，描述不同的流程状态。
* 增加时间轴侧边栏，显示更多详细信息。
* 增加时效与持续时间属性。  
* 增加时间轴与流程图联动效果。
